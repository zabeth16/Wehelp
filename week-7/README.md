## 變動性: 改名失敗
開光雞是在中南部比較常見的一種傳統特殊文化，藉由作法將霉運過度給雞，常見是廟宇、喪事。    
固定一定是使用這種白鳳雞種、甚至很多道士會割雞冠，但做法完的機會隨意丟棄，不可煮來吃。  
而這些雞通常很可憐，不是路殺就是被流浪動物給咬死。  
此文化當然是非常不好的QQ
